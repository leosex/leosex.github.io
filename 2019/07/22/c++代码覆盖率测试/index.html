c++代码覆盖率测试

2019-07-22 测试 c++ 评论

技术栈阐述

  1. 系统（我采用的是 macos 线上使用 centos7）
  2. cmake 版本 3.2 以上，既然人家一直更新就用最新的稳定版
  3. gcov 获取文本形式的覆盖率数据
  4. lcov 获取 html 形式的覆盖率数据

cmake 来配置代码覆盖率测试 Cmakelist.txt
